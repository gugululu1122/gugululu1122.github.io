出来逛逛

发表于 2017-02-23 | 分类于 点点滴滴 |

　　打算去信息楼听个讲座，结果没找到教室，离饭点又还早，于是背着书包转悠转悠着去图书馆旁边的草坪上散步去。
　　今天比昨几天稍稍暖和，阳光很好，晴空万里，好久没见过这么蓝的天了。天气真是不错，感觉校园里突然多出来很多老爷爷老奶奶，有一个人扶着老旧的自行车的，有女儿搀扶着的，还有坐着轮椅的。他们走走，就找个长椅坐一会儿，眯着眼睛，太阳真好啊。旁边有几只鸟儿，叽叽喳喳地从这根树枝上飞到那根树枝上。树枝光溜溜的，阳光一洒无余。
　　我慢慢走在鹅卵石道上。一块块凸起的鹅卵石，白色的，带点红色的，带点黄色的，还有黑色的，很是漂亮。鹅卵石旁边是青砖铺成的小道。不过比起小道来，我还是更喜欢走窄窄的鹅卵石路，踩着很舒服。
　　迎面走来一位大爷，我朝他笑了笑。他也朝我笑了笑。擦肩而过时，他把我叫住了，“姑娘你在干嘛？”我说在散步呢。他说好，那来长椅上坐会儿吧。问我是哪个学院的，几年级了，从哪儿来。我猜老人家可能是常在家，儿女在身边的时候少，可能很想有个聊天的伴儿吧。正好没什么事，就和他开始聊起来了。我说我是学经管的，老爷爷说，经管好啊，挣大钱的专业啊。我笑笑。接着他问既然是学商科的，那觉得现在企业有啥问题没有。我愕然，这该从何说起。不过陪老人家说话嘛，听还是最重要的。我刚要开口，老人家就说，他在家经常能接到诈骗电话啦，他买彩票又中奖了呀，他很警觉，觉得这些肯定是骗子，但是又没办法区分哪些电话是诈骗的，他好不接。他觉得现在人和以前大不同了，问我对这些诈骗经济现象有啥办法没。哦，原来老爷爷说的企业有问题是指诈骗啊。我脑子飞速转了一圈，正想多角度给他出点主意，老爷爷说你还是好好学习，出来之后做了大官好好治治。我又笑笑。
　　我问他是不是老师，他笑笑不说话，叫住了旁边一位在散步走来的老奶奶。他跟我说老奶奶是以前的教务处老师。我叫了一声奶奶好。老奶奶开玩笑说，你看你弄得人家小姑娘多不自在。两位老人打招呼的方式很特别，像小孩子一样比试了一招太极拳。我要走了，老爷爷说，下次常来啊。我点点头，说了一声好。
　　继续向前走，有奶奶在和小孙子玩捉迷藏的，有花式跳绳的，有围坐在一起聊天的，一个阳光正好的冬日下午。
